FAQ

Como salvo os arquivos no VS Code?

Crie um novo arquivo, cole o código, e salve com extensão .html ou .css. Mantenha os arquivos na mesma pasta.

O formulário envia mensagens de verdade?

Não — neste exemplo o envio é simulado via JavaScript. Para enviar de fato é necessário um backend ou serviço externo (ex.: Formspree).

Como animar a abertura do <details>?

Usamos CSS com transition em uma wrapper (.answer) para animar altura, opacidade e movimento ao abrir/fechar.
